validateHKCard

  • validateHKCard(idCard: JString): JBoolean

  • 验证香港身份证号码(存在Bug，部份特殊身份证无法检查)

    身份证前2位为英文字符，如果只出现一个英文字符则表示第一位是空格， 对应数字58 前2位英文字符A-Z分别对应数字10-35 最后一位校验码为0-9的数字加上字符"A"，"A"代表10

    将身份证号码全部转换为数字，分别对应乘9-1相加的总和，整除11则证件号码有效

    Parameters

    • idCard: JString

      身份证号码

    Returns JBoolean

    验证码是否符合
