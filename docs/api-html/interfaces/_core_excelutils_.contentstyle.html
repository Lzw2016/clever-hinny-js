Inherited from ExcelCellStyle.quotePrefix

Optional rotation

rotation: JShort

Inherited from ExcelCellStyle.rotation

设置单元格中文本的旋转度
注意：HSSF使用-90至90度的值，而XSSF使用0至180度的值。 此方法的实现将在这两个值范围之间进行映射， 但是，相应的getter返回此CellStyle所应用的当前Excel文件格式类型所要求的范围内的值。
